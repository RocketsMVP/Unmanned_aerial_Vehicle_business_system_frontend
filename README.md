# 重庆高速公路全域低空无人机AI应急处置业务系统 - 前端

> 基于 Vue 3 + TypeScript + Element Plus 构建的无人机应急处置可视化指挥平台

## 项目简介

本系统是 **重庆高速公路全域低空无人机AI应急处置业务系统** 的前端部分，聚焦重庆山区高速公路应急处置核心需求，提供"快速响应 - 精准研判 - 协同处置 - 复盘优化"的全流程可视化指挥平台。

系统实现交通事故、边坡滑坡、路面塌陷、隧道险情、危险品泄漏等突发事件的无人机快速响应与多部门协同处置。

## 核心功能模块

### 1. 应急快速响应模块

- **多渠道险情上报** - 支持自动触发（AI识别）、手动上报（APP/电话）、第三方上报（群众/养护人员）
- **无人机紧急部署** - GIS地图显示无人机实时位置/电量/状态，一键调度就近无人机
- **多无人机协同作业** - 自动分配作业任务（全景采集/细节拍摄/周边警戒）
- **应急飞行控制** - 自动规划作业路径，支持抗风/抗雾/抗雨稳定飞行

### 2. AI 应急态势智能感知模块

- **事故类型识别** - 交通事故/道路险情/隧道险情/危险品泄漏，识别准确率≥93%
- **现场信息研判** - 人员情况（被困人数/伤亡状态）、车辆情况（车型/损毁程度）、道路情况（拥堵范围/通航能力）
- **现场风险实时评估** - 二次事故风险、危险品风险、自然环境风险
- **态势报告自动生成** - 一键推送至指挥中心与救援人员

### 3. 多部门协同处置模块

- **应急资源可视化调度** - GIS三维地图实时显示救援车辆/人员/物资/医院位置与状态
- **最优路径自动规划** - 智能规划救援路径，避开拥堵与危险区域，响应时间≤2秒
- **跨部门协同联动** - 自动推送应急信息，支持救援人员实时反馈处置进度
- **远程指挥支持** - 实时回传无人机视频流，支持视频通话与语音指令

### 4. 处置过程管理与复盘模块

- **处置过程全程记录** - 完整记录险情上报/无人机部署/人员抵达/资源调用的全流程
- **现场数据实时更新** - AI动态更新态势分析结果，同步推送至所有协同部门
- **处置任务分配与跟踪** - 向各救援部门分配任务，实时跟踪完成状态
- **复盘分析与优化** - 自动生成复盘报告，支持多维度统计分析

### 5. 系统管理与保障模块

- **多角色权限管理** - 精细化角色配置（管理员/指挥人员/无人机操作员/救援人员/审核人员）
- **应急资源管理** - 救援车辆/人员/物资/医院信息的维护与状态实时更新
- **系统配置与适配** - 适配不同场景的无人机作业参数配置
- **应急保障功能** - 断网续传、数据加密存储、系统故障自动切换

## 快速开始

### 环境要求

- Node.js >= 18
- pnpm >= 8

### 安装依赖

```bash
pnpm install
```

### 开发环境启动

```bash
pnpm dev
```

### 构建生产版本

```bash
pnpm build
```

## 项目结构

```
src/
├── api/           # HTTP 请求封装（对接 FastAPI 后端）
├── assets/        # 静态资源（图片/图标/字体）
├── components/    # 公共组件（ProTable/地图组件/视频组件）
├── config/        # 全局配置
├── constants/      # 常量定义（险情类型/风险等级等）
├── directives/     # 自定义指令
├── hooks/         # 组合式函数（地图/视频/权限等）
├── languages/     # 国际化（中/英文）
├── layouts/      # 布局组件（指挥大屏/移动端等）
├── routers/       # 路由配置（静态/动态路由）
├── stores/        # Pinia 状态管理（无人机/态势/资源/任务）
├── styles/        # 全局样式与主题
├── typings/       # TypeScript 类型声明
├── utils/         # 工具函数（GIS/加密/日志）
└── views/         # 页面组件
    ├── emergency/      # 应急响应（险情上报/无人机调度）
    ├── perception/     # AI感知（态势识别/风险评估）
    ├── coordination/   # 协同处置（资源调度/任务跟踪）
    ├── management/     # 过程管理（记录/复盘）
    └── system/         # 系统管理（用户/角色/配置）
```

## 后端集成

系统对接 FastAPI 后端服务：

- Python FastAPI 服务
- SQLAlchemy 2.0 ORM
- Milvus 向量数据库
- MinIO 对象存储
- Redis 缓存
- MySQL 数据库

API 路径统一前缀 `/api/emergency/`。
