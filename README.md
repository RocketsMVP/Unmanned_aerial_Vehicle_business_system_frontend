# 劳务实名制与智能结算管理系统 - 前端

> 面向建筑工地的劳务工人全生命周期数字化管理平台

## 项目简介

DT_AI_Management_System 是一套完整的 **劳务实名制与智能结算管理系统** 前端应用，基于 Vue 3 + TypeScript + Element Plus 构建。

系统实现劳务工人从 **入场（实名认证、签合同）** → **在场（考勤管理）** → **离场（薪资结算）** 的全生命周期数字化管理。

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Vue 3.4+ / TypeScript |
| 构建工具 | Vite 5 |
| 状态管理 | Pinia |
| UI 组件库 | Element Plus 2.7+ |
| 路由 | Vue Router 4 |
| HTTP 客户端 | Axios |
| 图表 | ECharts 5 |
| 国际化 | Vue I18n |
| CSS 预处理器 | SCSS |

## 功能模块

### PC 端功能

- **人员档案及实名中心** - 人员管理、认证管理、状态流转
- **电子合同及签约中心** - 合同模板、签约发起、签约状态、合同归档
- **考勤与工时管理** - 考勤数据导入、工时报表生成
- **智能结算中心** - 结算单生成与确认、薪资报表与成本分析
- **项目现场管理** - 离场登记、实时在岗看板
- **培训管理** - 培训计划、培训记录
- **综合数据分析** - KPI 看板、趋势图
- **预警中心** - 预警管理与规则配置
- **系统管理** - 用户管理、组织管理、权限分配、操作日志、监管对接

### H5 端功能

- **我的认证** - 扫码激活、人脸实名认证、信息补全
- **我的合同** - 待签合同、合同签署、已签合同查阅
- **我的考勤** - 每日考勤记录查询
- **我的薪资** - 待确认结算单、薪资历史记录
- **个人中心** - 个人信息维护、消息通知

## 业务流程

```
预注册 → 扫码激活/工号查询 → 人脸实名认证 → 信息补全 → 已实名
                                                    ↓
                                            合同签约 → 已签约
                                                    ↓
                                            考勤打卡 → 已进场
                                                    ↓
                                            薪资结算 → 离场
```

## 快速开始

### 环境要求

- Node.js >= 18
- pnpm >= 8

### 安装依赖

```bash
pnpm install
```

### 开发环境启动

```bash
pnpm dev
```

### 构建生产版本

```bash
pnpm build
```

## 项目结构

```
├── config/       # 全局配置
├── constants/    # 常量定义

├── layouts/      # 布局组件
├── routers/      # 路由配置

├── styles/       # 全局样式
├── typings/      # TypeScript 类型
├── utils/        # 工具函数
└── views/        # 页面组件
```
