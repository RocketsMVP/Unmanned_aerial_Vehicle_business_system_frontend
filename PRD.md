## 一、产品概述

### 1.1 产品名称
劳务实名制与智能结算管理系统

### 1.2 产品目标
实现劳务工人从入场（实名认证、签合同）到在场（考勤管理）、到离场（薪资结算）的全生命周期数字化管理。

### 1.3 用户角色

| 角色 | 描述 | 使用终端 |
|------|------|----------|
| 管理员 | 项目管理人员，负责工人信息管理、合同发起、考勤审核、薪资结算 | PC端 |
| 班组长 | 班组负责人，管理本班组工人 | PC端 |
| 工人 | 劳务工人，实名认证、签署合同、查看考勤、确认薪资 | H5端 |

## 二、功能模块

### 2.1 PC端功能模块

#### 2.1.1 人员档案及实名中心

##### 人员档案
- **功能描述**：总览人员实名、合同、在岗状态，维护详细信息
- **核心功能**：
  - 人员列表展示（分页、筛选、搜索）
  - 新增人员（基本信息录入）
  - 编辑人员信息
  - 查看人员详情
  - 删除人员（软删除）
  - 批量导入/导出
- **字段信息**：姓名、身份证号、手机号、性别、年龄、工种、所属项目、所属班组、状态、头像等

##### 认证管理
- **功能描述**：查询人脸采集、活体检测核验记录，身份证手机信息登记，电子签名录入，提供人工审核通道
- **核心功能**：
  - 认证记录列表
  - 认证详情查看
  - 人工审核（通过/驳回）
  - 认证统计
- **认证流程**：人脸采集 → 活体检测 → 身份证OCR → 人工审核

##### 状态管理
- **功能描述**：管理状态流及黑名单
- **状态流转**：预注册(0) → 已实名(1) → 已签约(2) → 已进场(3) → 待离场(4) → 已离场(5)
- **核心功能**：
  - 状态变更操作
  - 黑名单管理（加入/移除）
  - 黑名单原因记录

#### 2.1.2 电子合同及签约中心

##### 合同模板
- **功能描述**：模板上传，可视化模板显示，版本控制
- **核心功能**：
  - 模板列表展示
  - 上传模板文件
  - 编辑模板信息
  - 删除模板
  - 版本管理
  - 模板预览

##### 合同发起
- **功能描述**：按项目、班组、个人批量发起签约任务，设置截止时间与逾期提醒
- **核心功能**：
  - 选择合同模板
  - 选择适用项目/班组/人员
  - 设置签署截止时间
  - 批量发起签约
  - 发送签约提醒

##### 签约状态
- **功能描述**：实时查看合同签署进度，查看已签署合同PDF及存证信息
- **核心功能**：
  - 合同列表（全部/待签/已签/已拒签）
  - 合同详情查看
  - 签署进度查看
  - PDF查看/下载
  - CA存证信息展示
  - 催办提醒

##### 合同归档
- **功能描述**：按项目、人员、时间等多维度检索合同，支持下载、作废
- **核心功能**：
  - 归档列表查询
  - 多维度检索
  - 批量下载
  - 合同作废

#### 2.1.3 考勤与工时管理

##### 考勤数据接入
- **功能描述**：支持Excel批量导入考勤数据，数据自动清洗与排重
- **核心功能**：
  - Excel批量导入
  - 数据格式校验
  - 自动清洗排重（5分钟内在同一设备重复打卡只记录第一次）
  - 跨天考勤处理（当日04:00至次日04:00）
  - 导入记录查询
  - 手工补卡
- **数据字段**：工人ID、设备ID、打卡时间、抓拍照片、来源
- **说明**：本系统不涉及门禁硬件设备对接，仅支持 Excel 导入考勤数据

##### 工时报表
- **功能描述**：生成个人/班组/项目维度工时汇总报表，支持自定义周期统计与导出
- **核心功能**：
  - 个人工时查询
  - 班组工时汇总
  - 项目工时统计
  - 自定义周期统计
  - 报表导出
- **统计维度**：正常工时、加班工时、缺卡、迟到、早退

#### 2.1.4 智能结算中心

##### 结算单生成与确认
- **功能描述**：基于工时生成待确认结算单，项目管理人员在线审核与调整，填写应发放金额，已发放金额，批量推送
- **核心功能**：
  - 结算单自动生成
  - 结算单手动调整
  - 项目管理人员审核
  - 金额确认
  - 批量推送H5
  - 工人确认/驳回
  - 发放状态跟踪
- **结算公式**：本期应发 = 正常工时×标准时薪 + 加班工时×加班时薪 + 奖金补贴 - 扣款罚款

##### 薪资报表与成本分析
- **功能描述**：生成项目人力成本月度报表、个人薪资历史记录查询、薪酬构成分析
- **核心功能**：
  - 项目成本报表
  - 个人薪资历史
  - 薪酬构成分析
  - 报表导出

#### 2.1.5 项目现场管理

##### 离场登记
- **功能描述**：更改在场状态，或根据业务规则，结算确认后，状态变为离场
- **核心功能**：
  - 离场办理
  - 状态校验（必须无未结清结算单）
  - 离场原因记录
  - 批量离场
  - **自动退场（长期未考勤自动触发）**：可配置 N 天未考勤自动触发离场

##### 在岗人员实时看板
- **功能描述**：大屏视图展示当日各项目/班组在岗人数、人员在场详情
- **核心功能**：
  - 实时在岗人数统计
  - 项目维度展示
  - 班组维度展示
  - 实时入场流水

#### 2.1.6 培训管理

##### 培训计划
- **功能描述**：创建和管理安全培训计划，支持按项目、班组、人员下发培训任务
- **核心功能**：
  - 培训计划创建
  - 培训计划编辑/删除
  - 培训计划下发

##### 培训记录
- **功能描述**：记录工人参加培训的情况，支持线上签到、线下补录
- **核心功能**：
  - 培训记录查询
  - 线下补录
  - 培训统计

#### 2.1.8 综合数据分析

##### 综合数据看板
- **功能描述**：展示核心KPI（总人数、实名率、合同签约率、在岗率），关键数据趋势图
- **核心功能**：
  - KPI指标卡片

#### 2.1.9 预警中心

##### 预警管理
- **功能描述**：对关键业务异常进行实时预警，支持预警提醒、处理、统计
- **核心功能**：
  - 预警列表展示

  - 预警处理/忽略
  - 预警统计报表

##### 预警规则

## 四、业务流程

### 4.1 人员入场流程
```
预注册 → 扫码激活/工号查询 → 人脸实名认证 → 信息补全 → 状态变更为"已实名"
```

### 4.2 实名认证流程
```
人脸采集 → 活体检测 → 身份证OCR识别 → 系统自动审核 → 人工审核（如需） → 认证通过/驳回
```

### 4.3 合同签约流程
```
管理员发起签约 → 工人收到通知 → 阅读合同全文 → 二次人脸验证 → 电子签名 → 签署成功 → 合同归档
```

### 4.4 考勤管理流程
```
设备打卡/手工补卡 → 数据导入 → 自动清洗排重 → 工时计算 → 考勤报表生成
```

### 4.5 薪资结算流程
```
考勤数据汇总 → 结算单自动生成 → 项目管理人员审核调整 → 批量推送H5 → 工人确认/驳回 → 确认后发放 → 状态变更
```

### 4.6 人员离场流程
```
管理员发起离场 → 校验是否存在未结清结算单 → 填写离场原因 → 确认离场 → 状态变更为"已离场"
```

## 九、非功能性需求

### 9.1 性能要求
- 接口响应时间 < 200ms（正常负载）
- 支持 1000+ 并发用户
- 发薪日需消息队列削峰

### 9.2 安全要求
- 身份证号加密存储（AES-256）
- 接口鉴权（JWT）
- 操作日志记录
- 数据隔离（公司→项目部→标段→班组）
  - 实现方式：基于组织架构的数据权限控制
  - 用户只能查看/操作所属组织及下级组织的数据
  - API 层统一注入数据隔离条件
